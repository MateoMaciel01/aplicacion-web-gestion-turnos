## 🏥 Aplicación Web de Gestión de Turnos - Clínica

Sistema web desarrollado para la gestión integral de una clínica médica, permitiendo administrar turnos, especialidades y usuarios mediante distintos roles.

---

## 🚀 Funcionalidades

### 👤 Paciente
- Visualización de especialidades médicas  
- Solicitud de turnos según disponibilidad  
- Cancelación y reprogramación de turnos  

### 🩺 Médico
- Visualización de turnos asignados  
- Actualización del estado de los turnos  
- Gestión del historial clínico de pacientes  

### ⚙️ Administrador
- Gestión completa de turnos  
- Alta de especialidades médicas  
- Administración de médicos y sus horarios  
- Reprogramación y cancelación de turnos  

---

## 🛠️ Tecnologías utilizadas

- C#  
- .NET / ASP.NET  
- SQL (MySQL / SQL Server)  
- HTML  
- CSS  

---

## 🗄️ Base de datos

El sistema utiliza una base de datos relacional para gestionar:

- Usuarios (pacientes, médicos, administradores)  
- Turnos  
- Especialidades  
- Historiales clínicos
